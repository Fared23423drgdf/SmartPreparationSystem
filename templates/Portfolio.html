طريقة تشغيل التطبيق

دليل تفصيلي لتشغيل النظام على جهازك

  1. تثبيت بايثون: تحقق من وجود بايثون عبر الأمر python --version (يتطلب نسخة 3.7 أو أحدث).
  2. تثبيت المتطلبات: شغل الأمر python -m pip install -r requirements.txt لتثبيت الحزم المطلوبة.
  3. تشغيل التطبيق: ابدأ التطبيق بالأمر python main.py.
  4. اختيار الكاميرا: اختر الكاميرا من القائمة المنسدلة (ستكون الأسماء واضحة).
  5. بدء المراقبة: اضغط Start لبدء التعرف على الوجوه.
  6. إضافة وجوه جديدة: استخدم زر Add New Face لتسجيل طلاب/موظفين جدد.
  7. عرض البيانات: اضغط Dashboard لرؤية سجلات الحضور وتصدير البيانات.
  8. إنهاء: أغلق النافذة عند الانتهاء - سيتم حفظ البيانات تلقائيًا.
ملاحظة: مجلد demo/ يحتوي على ملفات تجريبية غير مستخدمة في التطبيق الرئيسي.

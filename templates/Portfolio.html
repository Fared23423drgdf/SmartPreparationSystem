Tech Stack

Python 3.8, Tkinter, OpenCV, face_recognition and cv2-enumerate-cameras are used to build a local GUI application for attendance tracking. The dashboard is a static HTML page generated from the log file.

How to run

  1. Install requirements with pip install -r requirements.txt.
  2. Run python main.py to launch the GUI.
  3. Use the window to capture attendance.

    4. The old web demo is located in the demo/ folder and not part of the main application.
